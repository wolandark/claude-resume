Recommended Installation Method:

`curl https://raw.githubusercontent.com/wolandark/claude-resume/refs/heads/master/claude-resume > ~/.local/bin/claude-resume`

`chmod +x  ~/.local/bin/claude-resume`

**You Must Have `~/.local/bin` In Your `$PATH`**
